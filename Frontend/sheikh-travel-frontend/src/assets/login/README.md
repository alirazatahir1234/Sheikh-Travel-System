Place cinematic login assets in this folder:

- `world-map.png`
- `city.png`
- `bus.png`
- `van.png`
- `particles.png` (optional)

The login hero references these local files via:

- `assets/login/world-map.png`
- `assets/login/city.png`
- `assets/login/bus.png`
- `assets/login/van.png`

Using local assets avoids hotlink/CORS/expiry issues from external CDNs.
